# Pricing-Optimization-And-Discount-Impact-Analyzer
Data Analyst project using Python, SQL, Power BI

#Project Overview

This project analyzes the impact of discounts on sales and profitability in a retail dataset.
The objective is to identify the optimal discount range that maximizes profit while maintaining sales performance.

🔹 Problem Statement

Businesses often provide discounts to increase sales, but excessive discounts can reduce overall profitability.
This project aims to answer:

Do discounts always increase profit?
What is the optimal discount range?
Which products incur losses when discounted?

🔹 Tools & Technologies Used
Python (Pandas, Matplotlib) → Data Cleaning & Exploratory Data Analysis
SQL (MySQL) → Business Analysis & Querying
Power BI → Dashboard Creation & Data Visualization

🔹 Project Workflow
Data Loading and Cleaning using Python
Exploratory Data Analysis (EDA)
Business Analysis using SQL queries
Interactive dashboard creation in Power BI

🔹 Key Insights
Profit decreases significantly beyond ~30% discount, even though sales increase
Optimal discount range is between 10%–20%
Several products generate loss when discounted
Technology category shows higher profitability compared to others

🔹 Conclusion

This project demonstrates how data analysis can support pricing strategies by identifying discount levels that maximize profit and minimize losses.
